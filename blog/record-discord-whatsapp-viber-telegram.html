How to Record Discord, WhatsApp, Viber or Telegram Calls on Desktop

Method 1: OBS Studio (free, all four apps)

OBS Studio is a free open-source recorder. The steps below are the minimal setup — you don't need scenes, streaming config, or any of the video stuff. Just the audio output capture.

  1. Download and install OBS Studio from obsproject.com
  2. Open Settings → Output → Recording. Set a Recording Path and choose MKV or MP4 as the format
  3. Open Settings → Audio. Set Desktop Audio to your playback device (headphones or speakers). Optionally add your microphone under Mic/Auxiliary if you want to capture yourself too
  4. Close Settings. In the OBS main window, find the Desktop Audio level meter at the bottom. Play any audio — the meter should move. If it doesn't, right-click the meter → Properties and select the correct device
  5. Before your call: click Start Recording
  6. Make your call in Discord, WhatsApp Desktop, Viber, or Telegram Desktop as normal
  7. When done: click Stop Recording. The file saves to your output folder

No scenes or streaming setup needed. The audio level meter at the bottom of OBS is all you're using.

Method 2: Craig Bot (Discord voice channels only)

Craig records each person in a Discord voice channel on a separate audio track — useful if you want to edit speakers individually afterward. It only works in server voice channels, not private calls.

  1. Go to craig.horse and click "Add to Discord"
  2. Select your server from the dropdown and authorize Craig
  3. On your Discord server, join a voice channel
  4. In any text channel, type /join — Craig will join your current voice channel and post a download link in chat. Save it
  5. Record your session normally
  6. When done, type /stop. Craig leaves the channel
  7. Use the link from step 4 (or Craig's DM to you) to download the recordings. You'll get a separate audio file per speaker

Note: Craig doesn't work with Discord private calls (server voice channels only), and has no support for WhatsApp, Viber, or Telegram.

Method 3: Mono (all four apps, no per-call setup)

Mono records system audio directly from your computer's audio output. It doesn't join any call as a participant — completely invisible to everyone else. Works with Discord, WhatsApp Desktop, Viber, and Telegram Desktop without any per-app or per-call configuration. Transcription and speaker identification run locally on your device using Whisper.

  1. Download Mono and install it
  2. Open Mono, click Record
  3. Start your call in whichever app you're using — Discord, WhatsApp, Viber, or Telegram
  4. Talk normally. Mono captures everything playing through your computer
  5. When done, click Stop
  6. Mono transcribes the recording locally and shows you a searchable transcript with speaker labels

The same steps work for all four apps. No setup before each call. The free download allows one recording; $50 unlocks everything.
